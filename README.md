## ğŸš€ How to Run Locally

### Prerequisites

- **Node.js** (v14.0.0 or higher) - [Download here](https://nodejs.org/)
- **MongoDB** - [Download here](https://www.mongodb.com/try/download/community) or use MongoDB Atlas
- **npm** or **yarn** package manager

### Installation Steps

#### Backend Setup

**1. Clone the repository**
```bash
git clone https://github.com/yourusername/artwork-server.git
cd artwork-server
```

**2. Install backend dependencies**
```bash
npm install
```

**3. Create .env file in the root directory**
```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

**4. Start the backend server**
```bash
npm start
```
Server will run on `http://localhost:5000`

#### Frontend Setup

**1. Clone the frontend repository**
```bash
git clone https://github.com/yourusername/artwork-client.git
cd artwork-client
```

**2. Install frontend dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm start
```

**4. View the application**
- Open your browser and visit: `http://localhost:3000`
- Make sure the backend server is running

### Build for Production

#### Frontend
```bash
npm run build
```

#### Backend
```bash
npm run build
```
